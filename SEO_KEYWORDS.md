## Как читать таблицу

| Колонка | Смысл |
|---------|--------|
| **Показы/мес** | `totalCount` или частотность топ-фразы |
| **Intent** | C — compliance/инфо, M — коммерция, B — бренд/смешанный |
| **Приоритет** | P1 — страница/кампания в первую очередь, P2 — контент-хаб, P3 — длинный хвост |
| **Страница** | Рекомендуемый URL-сегмент на лендинге |

## Кластеры для контекстной рекламы

### Кампания «Журналы»

```
журнал технического обслуживания
журнал неисправностей
журнал учета неисправностей
электронный журнал учета оборудования
```

**Объявление:** Электронный журнал ТО и неисправностей — заявки, история, выгрузка для проверки.  
**Минус-слова:** огнетушител, лифт, автомобил, пожарн, скачать бесплатно без intent (тестировать)

### Кампания «ПО / 1С»

```
1с тоир
программа тоир
программа учета ремонтов
программа для обслуживания оборудования
```

**Объявление:** Masterdoc — проще 1С:ТОиР для сети объектов. Журналы, заявки, документы на оборудовании.

### Кампания «Холод»

```
журнал неисправностей холодильного
инструкция холодильного оборудования
```
