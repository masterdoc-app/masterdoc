# Wordstat × рынок: запросы и продукты

**Дата замера:** 2026-05-23 · **Регион:** РФ · **Источник:** Яндекс Wordstat (`totalCount`, ~30 дней)

**Как читать колонку «Рынок»:** типичные коммерческие решения и бренды в этой нише в РФ; где в Wordstat есть брендовый спрос — указан. Это **не** скрин выдачи Google/Яндекса по каждой фразе, а сопоставление intent → категория игроков.

**Intent:** C — compliance/образцы, M — покупка ПО, B — бренд/вход в сервис, N — шум/не наш ICP.

**Masterdoc:** ● сильный fit · ◐ частичный · ○ слабый / не наш трафик.

## Где нет готового продукта в поиске (голубой океан)

Запрос есть, устойчивого **категорийного** SaaS в Wordstat/выдаче нет:

| Запрос | Показов | Вывод |
|--------|--------:|-------|
| скачать инструкцию по эксплуатации | 15 782 | Дают файлы, не платформу |
| технические инструкции по эксплуатации | 8 240 | То же |
| приложение для сервисного инженера | 33 | Категория не названа |
| база знаний для сервисных инженеров | 0 | — |
| тоир документация / cmms документация | 70 / 0 | Не ищут связку |
| электронный журнал тоир | 6 | Не язык рынка |

**Позиция Masterdoc:** «инструкция у станка + журнал + заявка», не «электронный журнал ТОиР».
