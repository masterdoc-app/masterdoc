# База знаний MVP

**Дата:** 2026-05-20

## Выбранный источник

**Основной источник для индексации в Onyx:**

https://atlant.by/support/faq/

(альтернативный путь с локалью: https://atlant.by/ru/support/faq/)

## Почему этот источник

- Официальный FAQ производителя **ATLANT** (ЗАО «Атлант»).
- Темы под MVP приложения: установка и подключение, эксплуатация и уход, разморозка, типовые неисправности, настройка техники.
- Формат Q&A хорошо ложится на чат-ассистента: пользователь задаёт вопрос — ответ уже структурирован на сайте.

## Как подключить в Onyx

1. Коннектор **Web** с allowlist только на `atlant.by/support/faq/` (и при необходимости дочерние страницы FAQ).
2. **Document set** «Атлант FAQ» — привязать к persona/ассистенту в чате Masterdoc App.
3. Дополнительно (позже): PDF руководств через коннектор **File**, если понадобятся детали по конкретным моделям.

## Ограничения

- Перед публичным релизом — согласовать с правообладателем использование и индексацию контента с atlant.by.
- Не индексировать весь atlant.by без ограничений — только выбранный раздел FAQ.
